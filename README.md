# Clinical-Trial-Survival-and-Safety-Analysis-in-R-using-SDTM-ADaM-Standards

This project demonstrates an end-to-end clinical trial analysis workflow using R programming.  
The project is designed to simulate industry-level clinical programming work including:

- SDTM dataset creation
- ADaM dataset derivation
- Survival analysis
- Safety analysis
- Kaplan-Meier visualization
- Cox Proportional Hazard Model
- Tables, Listings, and Figures (TLFs)
- Metadata and define.xml concepts

The main goal of this project is to build strong practical skills in clinical data programming and statistical analysis using R.
